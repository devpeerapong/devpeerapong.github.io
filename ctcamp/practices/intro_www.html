พื้นฐาน www

World Wide Web

คือเครือข่ายเอกสารขนาดใหญ่ที่อยู่ที่สามารถเข้าถึงได้ผ่าน Internet ผ่านทาง Web Browser โดยการกำหนด URL ของที่เราต้องการ

ตัวละครหลักในโลก www

  1. URL
  2. Web Browser
  3. Web Server
  4. Web Page

URL (Universal Resource Locator)

คือ ที่อยู่ของข้อมูลที่อยู่บนเครือข่าย www ไม่ว่าจะเป็นไฟล์ HTML, รูป, เสียงหรือวิดีโอ โดยจะมาในรูปแบบ

protocal://domainname/path

เช่น

ที่อยู่ของ Website มจธ. คือ https://www2.kmutt.ac.th
ที่อยู่ของข้อมูลปริญญาโทใน Website มจธ. คือ www2.kmutt.ac.th/thai/pro_ungrd

Web Page

คือไฟล์เอกสารที่ส่วนใหญ่ถูกเขียนขึ้นด้วยภาษา HTML (Hyper Text Markup Language) ที่่สามารถเข้าถึงได้ผ่านทางเครือข่าย www

Web Browser

คือ Application ที่ใช้ในการเข้าถึงเครือข่าย WWW โดยจะรับ URL แล้วทำการส่ง Request ไปที่ ปลายทางนั้น เมื่อได้ Response กลับมาจะนำไฟล์ที่ได้มาแสดงผลให้เราเห็น โดย Web Browser ที่ ได้รับความนิยมคือ

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge

Web Server

คือ Program ที่ใช้ในการ Response ข้อมูลต่าง ๆ ที่มีการ Request มาจากต้นทางไม่ว่าจะเป็น

  • Web Page
  • ไฟล์รูป
  • ไฟล์เสียง
  • ไฟล์วิดีโอ

เกิดอะไรขึ้นเมื่อเราเข้า Web

  1. เปิด Computer
  2. เชื่อมต่อ Internet
  3. เปิด Web Browser
  4. พิมพ์ URL (http://www2.kmutt.ac.th/thai/pro_ungrd) ในช่องใส่ URL แล้วกด Enter
  5. Web Browser ส่ง Request ไปยัง Computer ที่มี Domain www2.kmutt.ac.th
  6. Computer ที่มี Domain www2kmutt.ac.th ที่ลง Web Server ไว้แกะกล่อง Request มาดู
  7. Web Server เจอ URL http://www2.kmutt.ac.th/thai/pro_ungrd แบ่งได้เป็น
    • Domain = www2.kmutt.ac.th
    • Path = /thai/pro_ungrd

  8. Web Server เข้าไปที่ folder /thai/pro_ungrd
  9. Web Server Response ไฟล์ index.html กลับมาให้ Computer เรา
  10. Web Browser บน Computer ของเรา Render ไฟล์ index.html ที่ได้รับให้เราเห็น
