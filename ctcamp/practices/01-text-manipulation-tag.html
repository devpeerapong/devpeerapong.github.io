นี่คือย่อหน้าที่ใช้สำหรับข้อความยาวๆ หนึ่งย่อหน้า เช่นเนื้อหาคำนำ เนื้อหาบทความต่าง

นี่คือตัวอย่างย่อหน้าที่ในไฟล์HTML.มีการเว้นบรรทัดด้วยปุ่ม Enter แต่ว่าเมื่อมาแสดงผล จะไม่มีการเว้นบรรทัด เพราะว่าหากจะให้เว้นบรรทัด ต้องใช้ TAG พิเศษที่เรียกว่า Tag br
เท่านั้นจำจะมีการเว้นบรรทัดได้

เพราะว่า Web Browser จะทำการมองช่องว่างต่างๆ เป็นแค่การเว้นวรรคหนึ่งครั้งเท่านั้น ทดสอบได้ด้วยการเว้นบรรทัดยาวๆ ในไฟล์ HTML.

และ TAG br ถือเป็น TAG พิเศษ ที่ไม่ต้องมี TAG ปิด เนื่องจาก TAG นี้ไม่ได้เน้นข้อความ แต่เป็น TAG ที่มีไว้เพื่อบอกว่า จุดนี้มีไว้เพื่อเว้นบรรทัดเท่านั้น

ข้อความต่างๆก็สามารถอยู้ได้โดยไม่มี TAG ครอบเช่นกัน แต่เราควรจะใช้ประโยชน์จาก TAG ต่างๆ มากกว่า

นอกจาก TAG สำหรับหัวข้อและข้อความแล้ว ยังมี TAG ช่วยต่างๆ เช่น TAG strong ที่จะทำให้ ข้อความที่ถูกครอบกลายเป็นตัวหนา
