Link Tag

link ถือเป็นส่วนสำคัญสำหรับเครือข่าย www เพราะมันช่วยให้เราเข้าถึง Website อื่น ๆ ได้อย่างง่ายดาย โดยส่วนใหญ่แล้วรูปแบบของ link จะมีไม่มาก เช่น

  1. link ไปยัง Website อื่น
  2. link ไปยัง Web Page อื่นของ Website เดียวกัน
  3. link ไปยัง ส่วนอื่นในหน้า Web Page เดียวกัน
  4. link ที่เปิดหน้าต่างใหม่

ในการทำให้ข้อความกลายเป็น link เราจะใช้ TAG a ครอบข้อความนั้น ๆ

แต่การใส่ TAG a อย่างเดียวไม่เพียงพอที่จะทำให้ข้อความกลายเป็น link ได้ เนื่องจากเราต้องกำหนดปลายทางให้กับ link นั้นด้วย

ซึ่งเราสามารถกำหนดปลายทางให้กับ link ได้ด้วยการใส่ คุณลักษณะ (Attribute) ที่ชื่อ href ให้กับ TAG a

การใส่ คุณลักษณะ (Attribute) เราจะใส่ในรูปแบบคือ

ชื่อคุณลักษณะ="ค่าที่เรากำหนด"

เช่น การกำหนดค่า คุณลักษณะชื่อ href ให้มีค่าเป็น http://google.co.th จะได้

href="http://google.co.th"

แล้วเราจะนำคุณลักษณะนี้ไปใส่ไว้ใน TAG เปิด ของ TAG a โดยจะเว้นวรรคออกมาจากชื่อ TAG 1 ครั้ง ดังนี้

<a href="http://google.co.th">

TAG a ข้างต้นจะทำให้ข้อความที่อยู่ภายใน TAG เป็น link ที่เชื่อมต่อไปยัง website ของ google

ตัวอย่าง TAG a ที่ link ไปยัง website ของ facebook

Go to Facebook

สำหรับการ link ไปยัง Web Page ที่อยู่ใน Website เดียวกัน (เช่น ภายในเครื่องของเรา) หากไฟล์นั้นอยู่ใน Folder เดียวกับไฟล์ที่มี link เราสามารถใส่ชื่อไฟล์ได้ทันที โดยหากเป็นชื่อไฟล์ ต้องใส่ทั้งชื่อและนามสกุลไฟล์ให้ครบถ้วนและถูกต้อง ทั้งพิมพ์เล็กพิมพ์ใหญ่

ตัวอย่าง TAG a ที่ link ไปยัง webpage อื่นใน website เดียวกัน

Go to Table Tag

สำหรับการ link ไปยัง Web Page ที่อยู่คนละ Folder จะใช้วิธี ยึดที่ ๆ ไฟล์นี้อยู่เป็นหลัก และหากไฟล์ที่ต้องการ link อยู่ใน Folder ลึกลงไป ให้ทำการลิ้งตามการเข้า Folder ในรูปแบบ ชื่อโฟลเดอร์/ชื่อไฟล์ หากไฟล์ที่ต้องการ link อยู่ที่ Folder ก่อนหน้า ให้ใช้ ../ (จุดสองตัว) เพื่อแทนการ Back กลับ Folder เช่น โครงสร้ายโฟลเดอร์ดังนี้

จาก myfile

  • ถ้าเราต้องการ link ไปยัง file1.html จะต้องใส่คุณลักษณะ (Attribute) href="file1.html"
  • ถ้าเราต้องการ link ไปยัง file2.html จะต้องใส่คุณลักษณะ (Attribute) href="/steam/file2.html"
  • ถ้าเราต้องการ link ไปยัง file3.html จะต้องใส่คุณลักษณะ (Attribute) href="/steam/dota/file3.html"
  • ถ้าเราต้องการ link ไปยัง file4.html จะต้องใส่คุณลักษณะ (Attribute) href="../file4.html"
  • ถ้าเราต้องการ link ไปยัง file5.html จะต้องใส่คุณลักษณะ (Attribute) href="../works/file5.html"

เราสามารถ link ไปหาข้อมูลในหน้าเดียวกันได้โดยการใส่ คุณสมบัติ (Attribute) ที่ชื่อ id ให้กับ TAG ที่เราต้องการจะ link ไป และเมื่อจะใส่ปลายทางที่ TAG a เราจะใช้ # นำหน้าตามด้วยชื่อ id เช่น หากต้องการจะ link ไปยัง TAG ที่มี id ชื่อ type จะเขียนได้ว่า href="#type" แล้วนำไปใส่ใน TAG a ดังนี้

Go to id="type"

เราสามารถกำหนดให้เมื่อกด link แล้วไปเปิด tab ใหม่ด้วยการใส่ คุณสมบัติ (Attribute) ที่ชื่อ target ให้มีค่าเป็น _blank เช่น
